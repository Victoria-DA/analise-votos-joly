# Montreal Election Data Analysis 📊

This project focus on analyzing the 2013 Montreal mayoral election results, specifically focusing on the performance of the candidate **Joly** across different districts. The analysis was performed using **Python**, **Pandas**, and **Plotly Express**.

## 🚀 Project Objective
The main goal was to practice data visualization and manipulation to answer specific business/analytical questions, such as identifying voting peaks and correlating them with the type of election result (Plurality vs. Majority).

## 🛠️ Technologies Used
* **Python**: Core programming language.
* **Pandas**: For data cleaning and manipulation.
* **Plotly Express**: For creating interactive bar charts and histograms.
* **Jupyter Notebook**: As the development environment.

## 📈 Key Findings
Based on the generated visualizations:
1. **Highest Vote Count**: The candidate Joly received the highest number of votes in the **141-Côte-de-Liesse** district.
2. **Analysis by Result Type**:
   - In **Plurality** systems, the highest voting district was also **141-Côte-de-Liesse**.
   - In **Majority** systems, the leading district for Joly was **121-La Pointe-aux-Prairies**.

## 📸 Visualizations
The project includes:
- **Interactive Bar Charts**: To compare absolute votes per district.
- **Histograms**: To analyze the distribution of votes categorized by the election outcome.

## 📋 How to Run
1. Clone this repository.
2. Install the requirements: `pip install pandas plotly`.
3. Open the `.ipynb` file in Jupyter Notebook or VS Code.
